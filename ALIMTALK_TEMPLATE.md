## 2. 템플릿 상세

### 템플릿 1 — 등원 알림

**템플릿명**: 학원_등원_알림

**내용**:
```
[#{학원명}] #{학생명} 학생이 #{시각}에 등원했습니다.
```

**변수 목록**:
| 변수명 | 설명 | 예시 |
|---|---|---|
| #{학원명} | 학원 이름 | 엘 영어학원 |
| #{학생명} | 학생 이름 | 홍길동 |
| #{시각} | 등원 시각 (KST, "오전/오후 H:MM" 12시간제) | 오후 3:05 |

---

## 3. 솔라피 연동 시 변수 매핑 (개발 참고)

> **주의**: 솔라피 API에 보내는 `variables` JSON의 **키는 `#{변수명}` 형태 그대로** 사용해야 한다 (템플릿 본문의 플레이스홀더와 1:1 매칭). 키에서 중괄호를 제거하면 치환되지 않고 원문이 그대로 발송된다.

```json
// 등원 발송 예시 (실제 코드: src/app/api/notify/route.ts)
{
  "templateId": "<SOLAPI_TEMPLATE_ID_CHECKIN>",
  "variables": {
    "#{학원명}": "엘 영어학원",
    "#{학생명}": "홍길동",
    "#{시각}": "오후 3:05"
  }
}

// 하원 발송 예시
{
  "templateId": "<SOLAPI_TEMPLATE_ID_CHECKOUT>",
  "variables": {
    "#{학원명}": "엘 영어학원",
    "#{학생명}": "홍길동",
    "#{시각}": "오후 5:30"
  }
}
```

## 4. 카카오 검수 유의사항

- 변수 형식은 반드시 `#{변수명}` 사용
- 광고성 문구 포함 시 즉시 반려 (예: "안심하세요", "믿을 수 있는" 등)
- 검수 통과 후 템플릿 내용 수정 불가 → 수정 필요 시 신규 템플릿 재등록
- 검수 소요 기간: 1~3 영업일

---

## 5. 검수 제출 순서

1. [카카오 비즈니스 채널 개설](https://center-pf.kakao.com)
2. 솔라피 회원가입 → 카카오 채널 연동
3. 위 템플릿 2개 검수 요청 제출
4. 검수 승인 후 솔라피 콘솔에서 비즈채널 발신 프로필 ID(`pfId`) 와 등원/하원 templateId 발급 → `.env.local`의 `SOLAPI_PFID` / `SOLAPI_TEMPLATE_ID_CHECKIN` / `SOLAPI_TEMPLATE_ID_CHECKOUT` 환경변수에 주입 (코드 하드코딩 금지)

## 6. SMS 자동 대체 발송 (운영 정보)

알림톡 발송 실패 시(카카오톡 미설치 / 차단 / 카카오 서버 장애 등) **동일 본문이 SMS로 자동 발송**된다. 학부모는 채널과 무관하게 등하원 알림을 받을 수 있다.

- 코드 위치: [src/app/api/notify/route.ts](src/app/api/notify/route.ts) — 솔라피 호출 시 `kakaoOptions.disableSms: false` 로 폴백 활성화
- 폴백 본문은 §2 템플릿 본문과 동일한 문구로 빌드 (학부모 혼선 방지)
- **운영 시 모니터링 포인트**: SMS는 알림톡 대비 단가가 높아 폴백이 빈번해지면 비용이 증가한다. 폴백 비율이 비정상적으로 높으면 카카오 채널 친구 추가율 / 차단율을 점검
